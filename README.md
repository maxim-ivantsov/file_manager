# 🖥️ OS Kernel Simulator

Учебный проект — симуляция базовых компонентов операционной системы на Python. Реализованы ядро с очередью процессов, файловая система и виртуальное устройство.

## 📸 Демонстрация

```
--- Меню ---
1. Создать файл
2. Прочитать файл
3. Взаимодействие с устройством
4. Запустить процессы
5. Выход

Выберите действие: 1
Введите имя файла: test.txt
Введите содержимое файла: Hello, World!

Выберите действие: 4
[FileSystem] Файл создан: device_files/test.txt
```

## 🏗️ Архитектура

```
├── main.py              # Точка входа, консольное меню
├── device_files/        # Папка для создаваемых файлов (генерируется автоматически)
└── kernel_logs.txt      # Лог всех операций (генерируется автоматически)
```

### Компоненты

| Класс | Описание |
|---|---|
| `Kernel` | Управляет очередью процессов, запускает потоки |
| `FileSystem` | Создание и чтение файлов в `device_files/` |
| `VirtualDevice` | Заглушка устройства — симулирует запись/чтение данных |

## ⚙️ Как работает

Действия 1–3 **не выполняются сразу** — они добавляют задачи в очередь (`queue.Queue`). Реальное выполнение происходит только при выборе пункта 4, где каждый процесс запускается через `threading.Thread` и ожидается через `join()`.

```
[Действие 1/2/3] → process_queue → [Действие 4] → Thread.start() → Thread.join()
```

## 🚀 Запуск

### Требования
- Python 3.x
- Сторонних библиотек не требуется

### Установка и запуск

```bash
git clone https://github.com/ВАШ_НИК/os-kernel-simulator
cd os-kernel-simulator
python main.py
```

## 📋 Использование

| Пункт меню | Действие |
|---|---|
| 1. Создать файл | Добавляет задачу записи файла в очередь |
| 2. Прочитать файл | Добавляет задачу чтения файла в очередь |
| 3. Устройство | Добавляет задачу взаимодействия с виртуальным устройством |
| 4. Запустить процессы | Выполняет все задачи из очереди последовательно |
| 5. Выход | Завершает программу и записывает финальную запись в лог |

## 🛠️ Стек технологий

- **Python 3** — основной язык
- **threading** — многопоточность для процессов
- **queue** — очередь задач
- **os / datetime** — работа с файловой системой и временем

## 📚 Чему учит проект
